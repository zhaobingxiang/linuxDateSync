# DateSync Release Notes

## v1.0.2 - 2026-06-26

### 🎯 同步计划优化

- **精确时间触发**: 修复定时同步触发逻辑，从双向容差匹配改为精确时间匹配，确保只在目标时间的整分钟触发
- **去重机制**: 添加 `sync_key` 去重机制，防止同一时间点重复触发同步
- **多时间点支持**: 月/周/日同步计划支持添加任意多个时间点
- **时间管理界面**: 从单组时/分输入改为列表式管理，支持添加、删除操作
- **列表显示优化**: 同步计划列表显示时用逗号拼接多个时间点

### 🔒 安全增强

- **SSH 认证安全**: 使用 `invoke_shell` 进行交互式 sudo 认证，替代明文密码传递，防止命令注入
- **密码加密存储**: 密码和 Root 密码采用基于机器标识的 Fernet 加密存储
- **临时文件安全**: 使用 `mktemp` 命令生成随机安全临时文件路径，防止符号链接攻击
- **文件解析优化**: 使用 `find -print0` 方式处理文件名，支持特殊字符（换行符等）

### 🛡️ 稳定性提升

- **线程安全**: 使用 Qt 信号/槽机制实现跨线程 UI 更新，避免子线程直接操作 Widget 导致崩溃
- **配置兼容性**: 新增 `_resolve_time_list()` 辅助函数，自动兼容旧版单值时间格式转为列表
- **字段名修复**: 修复配置保存时 `monthly_time/weekly_time/daily_time` 字段名不一致导致的程序崩溃
- **校时超时控制**: NTP 校时添加超时控制和多服务器重试机制
- **错误处理**: 增加 `SetLocalTime` 返回值检查，失败时记录详细错误信息

### 🖥️ 用户体验

- **对话框布局优化**: 将固定尺寸对话框改为自适应布局 + 滚动区域，解决切换同步类型后内容显示不全
- **月计划日期扩展**: 日期选择范围从 1-28 日扩展到 1-31 日，添加大小月提示
- **编辑不立即同步**: 更新配置时保留原 `last_sync` 时间，避免保存后立即触发同步
- **日志级别控制**: 支持运行时动态调整日志级别（DEBUG/INFO/ERROR）

### 🧹 代码质量

- **依赖清理**: 移除未使用的 `schedule` 包依赖
- **代码精简**: 删除重复导入、无效代码和未使用的类（`SyncWorker`）
- **注释优化**: 添加清晰的代码注释和文档字符串
